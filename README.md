
# mdpad

A way to write markdowns in a friendly way without having to learn markdown syntaxes. 

## Installation

```bash
> git clone https://github.com/joeSaad/mdpad.git
> npm start 
```

## Usage

just go to [http://mdpad.io](http://mdpad.io)
